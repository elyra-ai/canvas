# Test Harness

## Environment setup
- Nodejs `node 16.x`

#### Development setup
```sh
export NODE_ENV=development
./canvas_modules/common-canvas/build.sh
cd canvas_modules/harness
npm install
npm run build
npm start
```

#### Production setup

```sh
./canvas_modules/common-canvas/build.sh
cd canvas_modules/harness
npm install
npm build-prod
npm start-prod
```

#### Connect to canvas:
<http://localhost:3001>


## UI test

Ensure that the test harness is running.  The default location is http://localhost:3001.  
```sh
cd <base dir>/canvas_modules/harness
# Chrome
npx cypress run --headless --browser chrome
# Firefox
npx cypress run --headless --browser firefox
# Interactive
npx cypress open
```
